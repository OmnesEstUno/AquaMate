B — Soft advisor (recommended)

You can add anything. A live panel scores the whole tank — bioload, water envelope, aggression, schooling minimums — and explains every issue in plain language. Hard conflicts (salt vs fresh) still block; everything else is an override-able warning that teaches.

A third path exists: hybrid — hard-block only the impossible (water type, tank-too-small), soft-warn everything else. That's effectively B done carefully. Tell me which direction feels right.
