AKA pill — format decision

Show the single trade name that matched the current search — also known as: red neon tetra. Cardinal Tetra has 3+ trade names, but only surface the one that pulled it into the results. Prevents visual clutter and makes the "why is this here?" signal specific.

Hover teaser — mobile / touch fallback

Two options:

  • Long-press (250ms) shows teaser anchored below the card center, dismisses on tap-away. Native to touch and doesn't hijack the card's tap-to-navigate.
  • Skip on touch entirely — teaser is a hover-only affordance. Rationale: touch users go straight to detail pages where the summary is prominently displayed. No dual-code path needed.

Slight lean toward skip-on-touch for v2 simplicity; long-press can come later if user testing surfaces demand.

One color collision to note

The advanced care badge (#EF6C00 orange) and the Specialist advisory pill (#E65100 orange) sit in the same color family. On cards that carry BOTH, the two pills would look similar. In practice this pairing is uncommon (specialist species are almost all expert care), but flagging it — if it becomes an issue in the wild, we can pull specialist slightly warmer (deep amber) or advanced slightly cooler (rust).
